# 比特币铭文

**分类:** 夜间自动化  
**编号:** #15

## 🎯 痛点
- 比特币铭文（Inscription）操作复杂，需要等待区块确认
- 手动铭刻（Inscribe）需要持续监控交易状态
- 在合适的时间批量提交铭文任务可以节省费用

## ⚡ 核心功能
1. 批量创建和提交Ordinals铭文交易
2. 智能选择低费率时段广播交易
3. 实时监控铭文交易的区块确认状态
4. 支持铭文内容预览和验证
5. 完成后自动通知用户

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| bitcoin | 连接比特币网络 |
| ordi | 使用Ordinals协议工具进行铭文操作 |

## 📝 Prompt 模板
```
你是比特币Ordinals铭文助手。请帮助用户批量创建和管理比特币铭文。

功能要求：
1. 接收用户提供的铭文内容（文本、图片、JSON等）
2. 使用 ordi 工具创建铭文：
   ```bash
   ordi inscribe --fee-rate <费率> --content-type <类型> <文件>
   ```
3. 估算铭文费用（基于内容大小和费率）
4. 在用户确认后广播交易
5. 监控铭文状态变化，直到确认完成

注意事项：
- 建议在低费率时段执行（周末/凌晨）
- 先在测试网验证流程
- 确认钱包有足够余额支付铭文费用和gas
- 铭文一旦创建不可修改

请提供当前比特币网络费率建议，并执行铭文操作。
```
