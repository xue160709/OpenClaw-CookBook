# Trello 看板整理器

**分类:** 夜间自动化  
**编号:** #49

## 🎯 痛点
- Trello看板长期使用后变得越来越乱
- 卡片堆积过多，难以找到重点
- 需要定期整理但手动操作耗时

## ⚡ 核心功能
1. 自动归档已完成/过期的卡片
2. 按规则移动卡片到对应列表
3. 标记长期未动的卡片（建议处理或删除）
4. 清理空列表和废弃标签
5. 生成看板整理报告

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| trello | 连接Trello API管理看板 |
| 自动化 | 执行批量卡片操作 |

## 📝 Prompt 模板
```
你是Trello看板整理助手。请自动整理指定的Trello看板。

任务步骤：
1. 获取指定看板的所有列表和卡片：
   - 使用 Trello API: GET /1/boards/{boardId}/cards
   - 获取每个列表: GET /1/boards/{boardId}/lists
2. 分析卡片状态：
   - 已完成列表中超过30天未更新的卡片 → 归档
   - 超过60天未更新的卡片 → 标记为"待清理"
   - 没有标签的卡片 → 建议添加标签
3. 执行整理操作：
   - 归档符合条件的卡片: PUT /1/cards/{id}?closed=true
   - 移动卡片到对应列表
4. 清理看板：
   - 删除空的列表（可选）
   - 清理未使用的标签

生成整理报告：
```json
{
  "board_name": "...",
  "archived_cards": N,
  "moved_cards": N,
  "suggestions": [
    {"card": "...", "action": "..."}
  ],
  "stats": {
    "total_cards": N,
    "lists": N,
    "members": N
  }
}
```

注意：操作前备份看板数据。
```
