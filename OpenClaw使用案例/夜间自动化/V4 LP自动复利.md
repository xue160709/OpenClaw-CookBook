# V4 LP 自动复利

**分类:** 夜间自动化  
**编号:** #11

## 🎯 痛点
- Uniswap V4 流动性提供者需要手动定期复投收益，耗费精力
- 手动操作容易错过最佳复投时机，导致收益损失
- 多链DeFi操作复杂，需要持续关注gas费用

## ⚡ 核心功能
1. 自动监控指定流动性池的收益累积情况
2. 当收益达到设定阈值时自动触发复投操作
3. 智能选择低gas时段执行交易，节省成本
4. 支持多池同时监控和管理
5. 记录每次复投操作的详细日志，便于审计

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| ethereum | 连接以太坊网络读取链上数据 |
| defi | 调用Uniswap V4合约执行复投 |

## 📝 Prompt 模板
```
你是DeFi自动化助手，专注于Uniswap V4流动性自动复投。

请执行以下任务：

1. 查询指定LP池的当前状态（使用 Uniswap V4 Quoter 合约）
2. 计算预估收益和复投时机
3. 当收益超过阈值（如10美元）时：
   - 构造复投交易（调用 decreaseLiquidity + increaseLiquidity）
   - 估算最佳gas价格（使用 EIP-1559）
   - 发送交易并等待确认
4. 记录操作结果到日志文件

重要提醒：
- 交易前确认私钥已安全配置
- 主网操作需使用真实资产，测试网先验证
- 保留足够ETH支付gas
```
