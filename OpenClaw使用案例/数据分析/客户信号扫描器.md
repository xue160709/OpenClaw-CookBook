# 客户信号扫描器

**分类:** 数据分析  
**编号:** #23

## 🎯 痛点
在销售和客户关系管理中，需要及时发现潜在客户的购买信号（如社交媒体活跃度提升、产品需求表达），但人工追踪效率低，容易错过最佳跟进时机。

## ⚡ 核心功能
1. **多渠道监控** - 监控目标客户的社交媒体、邮件、CRM 等渠道
2. **信号识别** - 基于关键词和行为模式识别购买信号
3. **信号评分** - 根据信号强度进行优先级评分
4. **客户画像** - 整合客户多维度信息
5. **及时提醒** - 发现高价值信号时立即推送通知

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| web_fetch | 获取客户公开信息 |
| browser | 抓取社交媒体动态 |
| message | 推送客户信号提醒 |

## 📝 Prompt 模板
```
请帮我扫描以下客户的购买信号：
{customer_list}

扫描维度：
1. 社交媒体（Twitter/X、LinkedIn）最近30天动态
2. 产品相关关键词的提及（{product_keywords}）
3. 公司/个人重要事件（融资、产品发布、人员变动）
4. 竞品比较讨论

信号类型：
- 高价值信号：明确需求、预算提及、决策时间
- 中价值信号：兴趣表达、行业讨论
- 低价值信号：一般性关注

请返回：
1. 信号汇总（按客户分组）
2. 高价值信号详情
3. 跟进建议
```
