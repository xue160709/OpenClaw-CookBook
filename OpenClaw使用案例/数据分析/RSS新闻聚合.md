# RSS 新闻聚合

**分类:** 数据分析  
**编号:** #20

## 🎯 痛点
在信息爆炸时代，需要从多个信息源（博客、新闻网站、技术站点）持续获取最新资讯，但逐一访问效率低下，难以系统化地跟踪感兴趣的内容。

## ⚡ 核心功能
1. **RSS 订阅管理** - 管理多个 RSS 订阅源
2. **内容抓取** - 定期抓取各订阅源的最新内容
3. **智能过滤** - 基于关键词、标签过滤相关内容
4. **去重处理** - 识别并去除重复内容
5. **聚合推送** - 将筛选后的内容整理为每日/每周简报推送给用户

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| web_fetch | 获取 RSS 源内容 |
| message | 推送聚合简报 |

## 📝 Prompt 模板
```
请帮我从以下 RSS 源获取最新资讯：
{rss_sources}

筛选条件：
- 关键词：{keywords}
- 时间范围：最近24小时

请整理以下内容：
1. 热门文章列表（按来源分组）
2. 每篇文章的标题、摘要、原文链接
3. 推荐阅读（高相关度）
4. 生成今日资讯简报
```

## ⏰ Cron 配置
```json
{
  "schedule": "0 8 * * *",
  "task": "每日早间聚合 RSS 新闻并推送简报"
}
```
