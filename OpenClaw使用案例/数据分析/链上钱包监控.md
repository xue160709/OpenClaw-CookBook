# 链上钱包监控

**分类:** 数据分析  
**编号:** #22

## 🎯 痛点
在区块链投资和项目监控中，需要实时追踪特定钱包地址的交易活动（大户转账、项目方操作），但手动查询区块链浏览器效率低下，容易错过关键信息。

## ⚡ 核心功能
1. **钱包监控** - 监控指定钱包地址的链上活动
2. **交易追踪** - 实时获取转入/转出交易记录
3. **大额提醒** - 设置阈值，超过阈值的交易自动告警
4. **代币分析** - 分析持仓代币种类和数量变化
5. **趋势统计** - 统计交易频率、金额分布等数据
6. **多链支持** - 支持 ETH、BSC、Solana 等主流链

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| web_fetch | 调用区块链 API 获取链上数据 |
| browser | 访问区块链浏览器获取详细交易 |
| message | 推送大额交易告警 |

## 📝 Prompt 模板
```
请帮我监控以下钱包地址的链上活动：
{wallet_addresses}

监控要求：
1. 查询最近 24 小时的所有交易
2. 识别大额转账（单笔超过 {threshold} USDT）
3. 统计代币转入/转出情况
4. 分析地址活跃度（交易频率）
5. 检测异常行为（大额分散、突发大额转出等）

请返回：
1. 交易汇总表
2. 大额交易详情
3. 持仓变化分析
4. 风险预警（如有）
```
