# X 个人主页爬虫

**分类:** 数据分析  
**编号:** #08

## 🎯 痛点
在社交媒体分析和潜在客户挖掘场景中，需要批量收集 X (Twitter) 用户档案信息，但手动查找效率低下，难以系统化地获取用户的关键信息。

## ⚡ 核心功能
1. **用户档案抓取** - 通过浏览器或 API 获取目标用户的档案信息
2. **数据提取** - 提取用户名、简介、粉丝数、关注数、推文数等关键指标
3. **批量处理** - 支持多个用户名的批量抓取
4. **数据导出** - 将抓取的数据整理为结构化格式（JSON/CSV）

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| browser | 抓取 X 用户档案页面 |
| web_fetch | 获取用户公开信息 |

## 📝 Prompt 模板
```
请帮我抓取以下 X (Twitter) 用户的档案信息：
{username_list}

请提取并整理以下信息：
1. 显示名称和用户名
2. 个人简介
3. 粉丝数、关注数、推文数
4. 账户创建时间
5. 最近 pinned Tweet 内容（如果有）

请以表格形式呈现结果。
```

## ⏰ Cron 配置
```json
{
  "schedule": "0 */4 * * *",
  "task": "定期抓取目标用户档案更新数据"
}
```
