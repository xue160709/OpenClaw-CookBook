# 安全 CTF 课程

**分类:** 数据分析  
**编号:** #43

## 🎯 痛点
在网络安全学习和竞赛准备中，需要系统化地学习 CTF（Capture The Flag）各类题目，但学习资源分散，缺乏循序渐进的学习路径，难以量化学习进度。

## ⚡ 核心功能
1. **知识点梳理** - 整理 CTF 各方向（Web、Pwn、Crypto、Reverse、Misc）的核心知识点
2. **题目分类** - 按难度和知识点对历史 CTF 题目进行分类
3. **学习路径** - 生成适合不同水平的学习计划
4. **进度追踪** - 记录学习时长、完成题目数、技能提升
5. **能力分析** - 分析技能短板，推荐针对性练习

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| web_fetch | 获取 CTF 学习资源和题目 |
| message | 推送学习提醒和进度报告 |

## 📝 Prompt 模板
```
请帮我制定一个 CTF 学习计划：
1. 目标方向：{direction}（Web/Pwn/Crypto/Reverse/Misc）
2. 当前水平：{level}（入门/进阶/高级）
3. 每周可投入时间：{hours} 小时

请生成：
1. 推荐学习路线（按周分解）
2. 每个阶段需要掌握的核心知识点
3. 推荐的练习平台和题目
4. 学习资源清单（博客、工具、文档）
5. 阶段性自测题目
```
