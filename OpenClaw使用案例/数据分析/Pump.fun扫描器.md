# Pump.fun 扫描器

**分类:** 数据分析  
**编号:** #24

## 🎯 痛点
在 Pump.fun 平台进行 Token 交易时，需要快速发现新上线的潜力代币、分析代币数据（持有者、流动性、趋势），但手动刷页面效率低，容易错过早期机会。

## ⚡ 核心功能
1. **新币监控** - 实时监控 Pump.fun 新上线的代币
2. **数据抓取** - 获取代币的市值、持有者数量、交易量等关键指标
3. **趋势分析** - 分析代币价格走势和交易模式
4. **异常检测** - 识别大额买入/卖出、异常交易模式
5. **快速预警** - 发现符合条件的潜力币时立即推送

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| web_fetch | 获取 Pump.fun 平台代币数据 |
| browser | 抓取代币详细页面 |
| message | 推送新币预警 |

## 📝 Prompt 模板
```
请帮我扫描 Pump.fun 平台：
1. 获取最近1小时内上线的新代币
2. 筛选条件：
   - 初始市值 > $5000
   - 持有者数量 > 50
   - 24小时交易量 > $10000
3. 分析筛选出的代币：
   - 代币名称和符号
   - 当前市值和FDV
   - 持有者数量和分布
   - 资金池流动性
   - 价格走势（如果有）

请返回潜力币推荐列表和关键数据。
```
