# Moltbook 模式分析

**分类:** 数据分析  
**编号:** #25

## 🎯 痛点
在 Moltbook 平台上，用户需要分析交易模式、识别市场趋势，但平台数据可视化有限，难以深入挖掘历史数据中的规律和异常。

## ⚡ 核心功能
1. **交易数据抓取** - 获取指定账户的历史交易记录
2. **模式识别** - 使用统计方法识别重复的交易模式
3. **趋势分析** - 分析买入/卖出时机、频率、盈亏比例
4. **异常检测** - 识别异常大额交易、非活跃期后的活跃等
5. **可视化报告** - 生成交易模式分析图表和报告

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| web_fetch | 获取 Moltbook 账户数据 |
| browser | 抓取交易历史页面 |
| message | 推送分析报告 |

## 📝 Prompt 模板
```
请帮我分析 Moltbook 账户 {account_address} 的交易模式：
1. 获取最近 30 天的所有交易记录
2. 分析以下指标：
   - 总交易次数、买入/卖出比例
   - 平均持仓周期
   - 盈利/亏损交易占比
   - 交易频率分布（按小时/按天）
3. 识别以下模式：
   - 顺势交易还是逆势交易
   - 左侧交易还是右侧交易
   - 杠杆使用情况
4. 检测异常交易行为
5. 生成交易优化建议

请返回详细的分析报告。
```
