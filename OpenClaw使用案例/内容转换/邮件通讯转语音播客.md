# 邮件通讯转语音播客

**仓库:** awesome-openclaw-usecases-moltbook  
**分类:** 内容转换  
**编号:** #51

---

## 🎯 痛点

你订阅了很多有趣的通讯——行业更新、医学新闻、科技摘要——但很少有时间阅读。如果你的 AI Agent 能把这些邮件变成个人播客，在通勤、做饭或锻炼时收听呢？

---

## ⚡ 核心功能

1. **邮件监控** - Agent 检查收件箱，检测通讯
2. **内容提取** - 选择最有趣的 3-5 个故事
3. **脚本生成** - 写对话式播客脚本 (3-5 分钟阅读时间)
4. **语音生成** - 使用 TTS 文本转语音
5. **推送发送** - 作为语音消息发送到 Telegram

---

## 🛠️ 所需技能

| 技能 | 用途 |
|------|------|
| `email` | 读取收件邮件 |
| `tts` / ElevenLabs | 文本转语音 |
| `telegram` | 发送语音消息 |

---

## ⚙️ 前置条件

- Agent 可以访问的邮箱 (Gmail with app password)
- 连接到 OpenClaw 的 Telegram 或 Signal 频道
- (可选) ElevenLabs API 密钥用于更高质量的语音

---

## 📝 Prompt 模板

```
You are my personal podcast producer.

Every morning at 7 AM:
1. Check my email for newsletters received in the last 24 hours
2. Pick the most interesting 3-5 stories
3. Write a conversational podcast script (3-5 minutes reading time)
4. Convert to audio using TTS
5. Send the audio to me via Telegram

Style: Warm, conversational, like a friend catching you up over coffee.
Keep it under 5 minutes. Skip ads and promotional content.
```

---

## ⏰ Cron 配置

```
Schedule: 0 7 * * *
Action: Check email → Generate script → TTS → Send audio
```
