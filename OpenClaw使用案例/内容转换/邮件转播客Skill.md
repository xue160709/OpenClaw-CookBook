# 邮件转播客 Skill

**分类:** 内容转换  
**编号:** #38

---

## 🎯 痛点

你订阅了很多优质的邮件通讯（Newsletter），但没有时间逐一阅读。你希望有一个可复用的 AI Skill，能够把任何邮件通讯转换为语音播客，随时随地收听。这个 Skill 应该是独立的、通用的，可以处理各种类型的邮件内容。

---

## ⚡ 核心功能

1. **邮件内容提取** - 读取指定邮件或最新通讯
2. **智能摘要** - 提取关键信息和核心观点
3. **脚本转换** - 将邮件内容转换为适合播报的脚本
4. **语音生成** - 使用 TTS 转换为音频
5. **多平台推送** - 发送到 Telegram/飞书/Signal

---

## 🛠️ 所需技能

| 技能 | 用途 |
|------|------|
| `email` | 读取邮件内容 |
| `tts` / ElevenLabs | 文本转语音 |
| `telegram` / 飞书 | 发送语音消息 |

---

## 📝 Prompt 模板

```
你是一个通用的邮件转播客转换器。

功能：
1. 读取用户指定的邮件或最新收到的通讯邮件
2. 提取邮件中的核心内容和关键观点
3. 转换为 3-5 分钟的对话式播客脚本
4. 使用 TTS 转换为语音
5. 发送到用户指定的通讯渠道

规则：
- 保留关键数据和引用
- 去除广告和无关内容
- 使用通俗易懂的语言解释专业术语
- 保持轻松友好的对话风格
```

---

## ⏰ Cron 配置

```
# 定时运行（可选）
Schedule: 0 8 * * * 

# 也可手动触发
Action: 用户发送邮件 → 自动转换为播客发送回用户
```

---

## 💡 使用场景

- 将技术通讯转为播客收听
- 将行业报告转为语音摘要
- 将投资简报转为音频解读
