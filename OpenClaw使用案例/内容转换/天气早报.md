# 天气早报

**分类:** 内容转换  
**编号:** #03

---

## 🎯 痛点

每天出门前需要查看天气，但往往只是匆匆看一眼温度数字，不了解穿衣建议、出行注意事项等实用信息。如果 AI Agent 能在你起床时发送一份贴心的天气早报，包含穿衣指数、降水概率、紫外线强度等实用建议呢？

---

## ⚡ 核心功能

1. **天气数据获取** - 从天气 API 获取当前位置天气预报
2. **生活指数分析** - 穿衣指数、洗车指数、紫外线指数等
3. **早报生成** - 撰写温暖贴心的天气早报
4. **定时发送** - 每天早上 7 点自动发送

---

## 🛠️ 所需技能

| 技能 | 用途 |
|------|------|
| `weather` (wttr.in / Open-Meteo) | 获取天气预报 |
| `tts` | 语音播报（可选） |
| `telegram` / 飞书 | 发送早报 |

---

## 📝 Prompt 模板

```
你是我的天气早报员。

每天早上 7 点:
1. 查询广州今日天气（温度、湿度、降水概率、风力）
2. 获取穿衣指数、紫外线指数、空气质量
3. 生成 200 字的天气早报，包含：
   - 今日天气概况
   - 穿衣建议
   - 出行提醒
4. 发送到我的 Telegram

风格：温暖、贴心、像关心你的朋友。
```

---

## ⏰ Cron 配置

```
Schedule: 0 7 * * *
Action: 查询天气 → 分析指数 → 生成早报 → 发送
```
