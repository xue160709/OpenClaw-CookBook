# 奥运每日简报

**分类:** 内容转换  
**编号:** #02

---

## 🎯 痛点

奥运会期间，每天的比赛结果、奖牌榜更新、突破性表现等信息海量涌现。体育爱好者可能错过自己关注项目的关键赛事，或者没有时间追踪所有赛况。如果 AI Agent 能每天为你生成一份奥运精选简报呢？

---

## ⚡ 核心功能

1. **赛事数据获取** - 从奥运官网/新闻 API 获取当日赛果
2. **兴趣筛选** - 根据用户关注的运动项目筛选内容
3. **简报生成** - 撰写当日的奥运精选要点
4. **语音/图文推送** - 发送简报到 Telegram/飞书

---

## 🛠️ 所需技能

| 技能 | 用途 |
|------|------|
| `web_fetch` / `web_search` | 获取奥运资讯 |
| `tts` | 语音播报（可选） |
| `telegram` / 飞书 | 发送简报 |

---

## 📝 Prompt 模板

```
你是我的奥运专属简报员。

每天晚上 8 点:
1. 获取今日奥运会比赛结果（来自奥运官网或新闻源）
2. 筛选中国代表团表现和金牌赛果
3. 关注我喜欢的项目：游泳、田径、体操
4. 生成 300 字的奥运精选简报
5. 发送到我的 Telegram

风格：激情、专业、突出突破性表现。
```

---

## ⏰ Cron 配置

```
Schedule: 0 20 * * *
Action: 获取赛果 → 筛选 → 简报生成 → 发送
```
