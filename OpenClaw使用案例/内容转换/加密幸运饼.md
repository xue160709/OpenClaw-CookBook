# 加密幸运饼

**分类:** 内容转换  
**编号:** #05

---

## 🎯 痛点

加密货币市场波动剧烈，投资者经常感到焦虑和迷茫。你希望有一个有趣的 AI Agent，每天为你生成一条随机的「加密幸运饼」预言——可能是关于持币信心、市场心态的幽默提示，或者是一句充满哲理的rypto 小语。

---

## ⚡ 核心功能

1. **每日触发** - 定时触发或用户主动请求
2. **内容生成** - 生成一条有趣的加密幸运语
3. **视觉呈现** - 生成幸运饼风格的图片或 GIF
4. **推送发送** - 发送到 Telegram/飞书

---

## 🛠️ 所需技能

| 技能 | 用途 |
|------|------|
| `image` (DALL-E / Midjourney) | 生成幸运饼图片 |
| `telegram` / 飞书 | 发送幸运饼 |

---

## 📝 Prompt 模板

```
你是我的加密幸运饼。

每天下午 3 点:
1. 生成一条有趣的加密货币幸运语（20-50 字）
2. 风格：幽默、哲理、充满能量
3. 示例：
   - "今天持币不动，就是最好的交易。"
   - "别盯着 K 线发愁，起来喝水。"
   - "你的币可能还在套牢，但你的心态已经自由。"
4. 生成一张幸运饼风格的图片
5. 发送到我的 Telegram

注意：根据当天市场情绪调整语气。市场大跌时给信心，大涨时提醒风险。
```

---

## ⏰ Cron 配置

```
Schedule: 0 15 * * *
Action: 生成幸运语 → 生成图片 → 发送
```

---

## 💡 扩展玩法

- **开饼功能** - 用户可以"打开"幸运饼获得当日建议
- **收藏功能** - 将幸运语保存到表格记录
