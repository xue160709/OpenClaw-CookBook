# 医学邮件转播客

**分类:** 内容转换  
**编号:** #01

---

## 🎯 痛点

医学研究人员、临床医生和医疗从业者每天会收到大量医学期刊摘要、临床试验更新、FDA 审批通知等专业邮件。由于工作繁忙，往往无法及时阅读这些重要资讯。如果 AI Agent 能将这些医学邮件转化为语音播客，在通勤或休息时收听最新医学进展呢？

---

## ⚡ 核心功能

1. **邮件监控** - 定期检查医学相关邮件（NEJM、Lancet、FDA 等通讯）
2. **内容筛选** - 提取关键临床发现、药物审批、指南更新
3. **医学播客脚本** - 撰写适合语音播报的医学摘要（保留关键数据）
4. **语音生成** - 使用 TTS 转换为音频
5. **推送发送** - 发送到 Telegram/飞书

---

## 🛠️ 所需技能

| 技能 | 用途 |
|------|------|
| `email` | 读取医学邮件 |
| `tts` | 文本转语音 |
| `telegram` / 飞书 | 发送语音消息 |

---

## 📝 Prompt 模板

```
你是我专属的医学播客制作人。

每天早上 7 点:
1. 检查收件箱中的医学期刊通讯（NEJM、Lancet、JAMA、FDA 更新等）
2. 筛选最重要的 2-3 条医学进展
3. 撰写 3-5 分钟的医学播客脚本，包含：研究结论、关键数据、患者影响
4. 使用 TTS 转换为语音
5. 发送到我的 Telegram

风格：专业但易懂，保留关键医学数据，解释专业术语。
```

---

## ⏰ Cron 配置

```
Schedule: 0 7 * * *
Action: 医学邮件 → 筛选 → 脚本 → TTS → 发送
```
