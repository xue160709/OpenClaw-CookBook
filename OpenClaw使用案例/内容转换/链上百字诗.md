# 链上百字诗

**分类:** 内容转换  
**编号:** #04

---

## 🎯 痛点

你有一些简短的思想、感悟或诗句，希望永久保存在区块链上，成为数字时代的「时间胶囊」。传统的链上铭刻需要技术知识，你希望用自然语言就能把俳句（Haiku）或短诗 inscription 到比特币网络上。

---

## ⚡ 核心功能

1. **内容输入** - 接收用户输入的俳句/短诗
2. **格式验证** - 检查是否符合 Haiku 格式（5-7-5）或字数限制
3. **链上铭刻** - 调用比特币铭刻 API 将内容写入区块链
4. **确认反馈** - 返回链上交易 ID 和铭刻证明

---

## 🛠️ 所需技能

| 技能 | 用途 |
|------|------|
| `ordinals` / 铭刻工具 | 链上铭刻 |
| `telegram` / 飞书 | 接收输入、发送确认 |

---

## 📝 Prompt 模板

```
你是我的链上诗人。

功能：
1. 接收用户发送的俳句（5-7-5）或短诗（最多 100 字）
2. 验证格式是否符合要求
3. 将内容铭刻到比特币Ordinals链上
4. 返回交易 ID 和链上链接

规则：
- 只接受诗歌或简短感悟内容
- 过滤不当内容
- 铭刻完成后告知用户 TXID 和查看链接
```

---

## ⏰ Cron 配置

```
# 手动触发
Action: 用户发送诗句 → 验证 → 铭刻 → 返回 TXID
```

---

## 💡 扩展玩法

- **每日一诗** - 每天自动铭刻一条用户创作的俳句
- **纪念日铭刻** - 将特殊日期的感悟永久保存
