# Telegram 智能家居

**分类:** 日常生活  
**编号:** #62

## 🎯 痛点
- 人在外面想远程控制家中设备
- 各类智能家居App太多，操作麻烦
- 忘记是否关闭了某些设备

## ⚡ 核心功能
1. 远程控制：通过Telegram控制智能家居设备
2. 状态查询：随时查看家中设备状态
3. 场景联动：一键执行预设场景（离家/回家/睡眠）
4. 异常提醒：设备异常或传感器报警时通知

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| feishu-im-user-message | Telegram消息交互 |
| nodes | 控制智能设备（如通过HomeKit等） |
| feishu-task | 创建设备异常提醒任务 |

## 📝 Prompt 模板
```
你是我的智能家居助手，通过Telegram与我交互。

帮我管理家中的智能设备：
1. 我可以通过Telegram发送命令控制设备，例如：
   - "开灯" / "关灯"
   - "打开空调到26度"
   - "开启离家模式"
   - "查看所有设备状态"
2. 每天晚上10点自动检查并汇报家中设备状态
3. 如果有设备异常（如传感器检测到异常），第一时间通知我
4. 支持场景模式：
   - 离家模式：关闭所有灯、空调设为节能
   - 回家模式：开灯、开空调、拉开窗帘
   - 睡眠模式：调暗灯光、关闭不必要的电器
5. 记录我的使用习惯，逐渐优化自动化

目前支持的设备：智能灯、空调、智能插座、窗帘电机、门窗传感器。
```

## ⏰ Cron 配置
```json
{ "schedule": "0 22 * * *", "task": "smart_home_status_check" }
```
