# 智能日历提醒

**分类:** 日常生活  
**编号:** #55

## 🎯 痛点
- 日程太多，容易忘记重要会议/预约
- 常规日历提醒不够智能，无法根据实际情况调整
- 缺少提前准备提醒（如需要提前准备资料）

## ⚡ 核心功能
1. 智能分级：根据日程重要性自动分级提醒
2. 提前准备提醒：会议前N小时提醒准备资料
3. 交通时间计算：自动计算通勤时间并提醒出发
4. 冲突检测：检测时间冲突并提醒调整

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| feishu-calendar | 读取日历日程 |
| feishu-task | 创建准备事项任务 |
| weather | 结合天气计算出行时间 |

## 📝 Prompt 模板
```
你是我的智能日历助手。

每天晚上扫描我的飞书日历，帮我做以下事情：
1. 列出明天所有日程，按重要程度排序
2. 对于重要的会议/预约，提前提醒我准备相关资料
3. 计算从家/当前位置到会议地点的通勤时间，给出建议出发时间
4. 检测是否有时间冲突，提前预警
5. 如果明天有重要的外勤，提前查看天气并提醒携带雨具等

提醒风格：简洁、实用、提前量足够。
```

## ⏰ Cron 配置
```json
{ "schedule": "0 20 * * *", "task": "calendar_preview_tomorrow" }
```
