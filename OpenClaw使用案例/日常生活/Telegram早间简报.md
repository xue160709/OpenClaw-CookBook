# Telegram 早间简报

**分类:** 日常生活  
**编号:** #52

## 🎯 痛点
- 早上匆忙出门，无法快速获取当天重要信息
- 需要手动整理天气、新闻、日程等多个来源的信息
- 容易错过重要的待办事项和会议

## ⚡ 核心功能
1. 天气摘要：获取当前位置的天气情况，包含温度、降雨概率、空气质量
2. 日程概览：读取当天日历，列出重要会议和日程安排
3. 待办提醒：汇总当天的关键待办事项
4. 新闻精选：聚合2-3条重要新闻或行业动态
5. 自定义问候：根据时间显示早安/午安等问候语

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| feishu-calendar | 读取当天日程安排 |
| weather | 获取天气预报信息 |
| feishu-task | 读取待办任务列表 |
| feishu-im-read | 通过Telegram发送早间简报 |

## 📝 Prompt 模板
```
你是我的早间生活助手。每天早上7:30通过Telegram发送早间简报给我。

请帮我整合以下信息：
1. 今天的天气情况（温度、是否下雨、空气质量）
2. 今天的日程安排（会议、活动等）
3. 今天的待办事项（重要的3-5条）
4. 可选的新闻摘要或提醒

请用简洁、温暖的语气整理成早间简报，结尾可以加上一句励志或轻松的话。
```

## ⏰ Cron 配置
```json
{ "schedule": "0 7 * * *", "task": "send_morning_briefing" }
```
