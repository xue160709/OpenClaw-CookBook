# 社交媒体监控

**分类:** 日常生活  
**编号:** #64

## 🎯 痛点
- 想知道某些人/话题的最新动态但不想刷社交媒体
- 重要信息容易被淹没在信息流中
- 想要有针对性地关注特定内容

## ⚡ 核心功能
1. 关键词监控：监控特定关键词/话题
2. 人物动态：关注特定博主的更新
3. 竞品动态：监控竞品或行业动态
4. 摘要报告：定期汇总监控内容

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| web_search | 搜索社交媒体动态 |
| feishu-im-user-message | 推送监控摘要 |
| feishu-bitable | 存储监控配置和历史记录 |

## 📝 Prompt 模板
```
我是你的社交媒体监控助手。

帮我监控以下内容：
1. 监控列表（飞书多维表格管理）：
   - 关注对象（人/品牌/公司账号）
   - 平台（Twitter/X/微博/小红书/Instagram等）
   - 关键词（想特别关注的话题）
   - 更新频率要求
2. 每天下午5点汇总以下信息：
   - 关注对象的最新动态（按重要性排序）
   - 关键词相关的新内容
   - 是否有需要我立即知道的信息
3. 周末生成一份本周社交媒体动态报告
4. 当监控对象有重要更新时（如发布重要产品/重大新闻），立即单独通知我
5. 支持新增/删除/修改监控对象

监控原则：只监控对我有价值的，避免信息过载。
```
