# 新闻摘要聚合

**分类:** 日常生活  
**编号:** #59

## 🎯 痛点
- 信息过载，没时间看所有新闻
- 各类新闻App太杂，难以聚焦
- 想要个性化定制的新闻简报

## ⚡ 核心功能
1. 主题订阅：按兴趣领域订阅新闻源
2. 智能摘要：自动提取新闻关键信息
3. 定时推送：每天固定时间推送新闻简报
4. 深度阅读：保留原文链接供深入阅读

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| web_search | 搜索热点新闻 |
| web_fetch | 获取新闻详细内容 |
| feishu-im-user-message | 推送新闻摘要 |

## 📝 Prompt 模板
```
你是我的新闻摘要助手。

每天为我聚合新闻简报：
1. 我感兴趣的领域：科技、商业、设计、AI/机器学习
2. 每天早上8点推送新闻摘要，包含：
   - 最重要的3条科技/商业新闻（附一句话概括）
   - 1条设计或AI领域的深度资讯
   - 1条可能对我有用的工具或资源
3. 每条新闻包含：标题、来源、核心要点、一句话点评
4. 附上原文链接供深入阅读
5. 周末可以适当增加篇幅，做一周新闻回顾

新闻筛选标准：对我有用的、有启发性的、避免标题党。
```

## ⏰ Cron 配置
```json
{ "schedule": "0 8 * * *", "task": "news_daily_digest" }
```
