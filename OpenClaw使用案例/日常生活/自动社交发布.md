# 自动社交发布

**分类:** 日常生活  
**编号:** #65

## 🎯 痛点
- 想在多个平台发布内容但操作繁琐
- 忘记在最佳时间发布
- 不同平台需要不同的格式调整

## ⚡ 核心功能
1. 定时发布：预设发布时间自动发布
2. 多平台适配：自动调整内容格式适配不同平台
3. 内容库：建立可复用的内容素材库
4. 发布日历：规划发布排期

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| feishu-bitable | 管理和排期发布内容 |
| browser | 自动化发布到各社交平台 |
| web_search | 获取热点话题供内容策划 |

## 📝 Prompt 模板
```
你是我的社交媒体自动发布助手。

帮我管理多平台内容发布：
1. 在飞书多维表格中建立内容日历：
   - 内容文案（各平台版本）
   - 配图/素材
   - 发布平台（Twitter/微博/小红书/Instagram等）
   - 发布时间
   - 发布状态（草稿/已发布/待发布）
2. 我提供一条原创内容后，帮我：
   - 调整成各平台适合的格式
   - 排入合适的发布时间
   - 提前提醒我确认
3. 每天检查待发布内容，确保按时发出
4. 发布后记录发布时间和平台反馈
5. 每月分析各平台表现最好的内容类型

发布平台：Twitter、微博、小红书、Instagram（暂定）
```

## ⏰ Cron 配置
```json
{ "schedule": "0 8 * * *", "task": "check_daily_posts" }
```
