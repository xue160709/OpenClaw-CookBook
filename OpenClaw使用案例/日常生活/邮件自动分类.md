# 邮件自动分类

**分类:** 日常生活  
**编号:** #56

## 🎯 痛点
- 邮件太多，找不到重要邮件
- 各类订阅邮件混杂在收件箱
- 手动分类太耗时

## ⚡ 核心功能
1. 智能分类：自动识别邮件类型（工作/个人/订阅/通知）
2. 重要邮件标记：高亮重要邮件（来自关键联系人）
3. 自动归档：定期归档旧邮件
4. 周报汇总：每周生成邮件处理报告

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| feishu-bitable | 存储邮件分类规则和统计 |
| feishu-im-user-message | 发送邮件摘要通知 |

## 📝 Prompt 模板
```
你是我的邮件管理助手。

请帮我自动化处理邮件：
1. 扫描收件箱，按以下规则自动分类：
   - 工作邮件：来自工作域名，主题包含项目/会议关键词
   - 重要邮件：来自老板、客户、合作伙伴
   - 订阅邮件：来自常见订阅服务（新闻、电商等）
   - 通知邮件：系统通知、账单、物流信息
2. 每天上午10点给我发一封邮件摘要，包含：
   - 重要未读邮件（3-5封）
   - 待处理邮件数量
   - 需要立即关注的邮件
3. 每周六清理并归档30天前的已读邮件
4. 每月生成邮件处理统计报告

注意：仅处理Gmail/QQ邮箱等支持的邮箱。
```

## ⏰ Cron 配置
```json
{ "schedule": "0 10 * * *", "task": "email_daily_digest" }
```
