# 每日学习日志

**分类:** 日常生活  
**编号:** #57

## 🎯 痛点
- 学习内容零散，缺乏系统性记录
- 学完就忘，无法形成知识体系
- 需要回顾时找不到之前的笔记

## ⚡ 核心功能
1. 学习记录：记录每天学习的内容、来源、关键要点
2. 知识卡片：自动提取关键概念生成知识卡片
3. 定期复习：基于艾宾浩斯遗忘曲线提醒复习
4. 统计报告：每周/月生成学习时长和进度报告

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| feishu-bitable | 存储学习记录和知识卡片 |
| feishu-create-doc | 创建学习日志文档 |
| feishu-task | 创建复习提醒任务 |

## 📝 Prompt 模板
```
你是我的学习成长助手。

请帮我建立每日学习日志系统：
1. 每天学习结束后，引导我记录：
   - 今天学了什么？（主题/知识点）
   - 从哪里学的？（课程/书籍/文章/视频）
   - 3个最核心的要点是什么？
   - 我计划怎么用这些知识？
2. 自动从记录中提取关键概念，存入知识卡片库
3. 根据艾宾浩斯遗忘曲线，在学习后1天、3天、7天、14天提醒我复习
4. 每周六生成本周学习报告：
   - 学习时长统计
   - 主题分布
   - 最值得回顾的内容
5. 每月推荐一门可能感兴趣的新课程或书籍

记录风格：简洁、 actionable、有深度。
```

## ⏰ Cron 配置
```json
{ "schedule": "0 21 * * *", "task": "daily_learning_journal" }
```
