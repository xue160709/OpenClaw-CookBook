# 冷关系复活

**分类:** 日常生活  
**编号:** #54

## 🎯 痛点
- 很长时间没联系的朋友/同事，关系变淡
- 不知道如何自然地重启对话
- 担心突然联系显得尴尬

## ⚡ 核心功能
1. 关系梳理：记录联系人列表及最后联系时间
2. 智能提醒：标记超过一定时间未联系的人
3. 话术建议：根据关系亲密度提供合适的开场白
4. 节日关怀：在生日、节日等特殊日子提醒问候

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| feishu-bitable | 存储联系人关系和联系记录 |
| feishu-search_user | 搜索联系人信息 |
| feishu-im-user-message | 发送复活问候消息 |

## 📝 Prompt 模板
```
你是我的关系维护助手。

请帮我管理人际网络：
1. 在飞书多维表格中记录重要联系人，标注最后联系时间
2. 每周提醒我需要"复活"的冷关系（超过30天未联系）
3. 为每个人提供3个可选的开场话题：
   - 最近的共同经历/话题
   - 对方的近况关心
   - 分享一个有价值的资讯/资源
4. 当对方回复后，帮我记录新的联系时间

关系维护原则：不群发、真诚、有价值。
```

## ⏰ Cron 配置
```json
{ "schedule": "0 10 * * 6", "task": "cold_relationship_check" }
```
