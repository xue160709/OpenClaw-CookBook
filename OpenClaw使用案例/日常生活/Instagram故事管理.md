# Instagram 故事管理

**分类:** 日常生活  
**编号:** #53

## 🎯 痛点
- 想要定期发布Instagram Story但经常忘记
- 需要手动整理照片/视频素材
- 不知道什么时间段发布效果最好

## ⚡ 核心功能
1. 内容策划：预设每周Story主题和内容脚本
2. 素材管理：整理手机相册中的照片/视频
3. 自动发布：定时发布预编辑好的Story
4. 效果追踪：记录每条Story的观看数据

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| browser | 自动化操作Instagram网页版 |
| feishu-bitable | 存储内容日历和发布记录 |
| nodes | 调用手机摄像头拍照 |

## 📝 Prompt 模板
```
你是我的Instagram Story运营助手。

请帮我：
1. 每周日规划下周7天的Story主题日历（存入飞书多维表格）
2. 在发布日自动提醒我准备素材
3. 帮我编辑Story文案（简洁、有吸引力）
4. 记录每次发布后的观看数据

内容风格要求：真实、有生活气息、偶尔带有个人特色。
```

## ⏰ Cron 配置
```json
{ "schedule": "0 18 * * *", "task": "instagram_story_reminder" }
```
