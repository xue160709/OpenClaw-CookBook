# 会议笔记生成

**分类:** 日常生活  
**编号:** #68

## 🎯 痛点
- 会议中来不及记录所有要点
- 事后整理笔记耗时
- 会议结论和待办容易遗漏

## ⚡ 核心功能
1. 会议记录：支持语音/文字实时记录会议
2. 自动摘要：会后自动生成会议纪要
3. 待办提取：自动识别并列出会议中的待办事项
4. 归档整理：按会议主题归档到知识库

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| feishu-calendar | 获取会议信息 |
| feishu-create-doc | 创建会议纪要文档 |
| feishu-task | 从会议中提取待办任务 |
| openai-whisper | 语音转文字 |

## 📝 Prompt 模板
```
你是我的会议助手。

帮我高效管理会议：
1. 会前：
   - 自动读取飞书日历，获取会议主题、参与人、会议链接
   - 提醒我提前准备相关资料
2. 会中：
   - 我可以语音输入会议要点，你会帮我记录
   - 实时帮我标注待办事项和重要决策
3. 会后（自动触发）：
   - 生成结构化会议纪要：
     - 基本信息（时间、参与人、主题）
     - 讨论要点（按主题分组）
     - 决策结论（加粗高亮）
     - 待办事项（负责人 + 截止时间）
   - 将待办事项同步到飞书任务
   - 整理成飞书云文档保存
4. 待办跟踪：
   - 在截止日期前提醒负责人
   - 会议后3天回访确认进展
5. 定期汇总：每月生成本月会议统计（数量、时长、待办完成率）

会议纪要风格：简洁、结构化、action-oriented。
```

## ⏰ Cron 配置
