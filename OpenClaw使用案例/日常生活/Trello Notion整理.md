# Trello/Notion 整理

**分类:** 日常生活  
**编号:** #66

## 🎯 痛点
- Trello和Notion数据分散，难以统一管理
- 看板/页面太多，找不到需要的内容
- 跨平台信息需要手动同步

## ⚡ 核心功能
1. 统一索引：建立Trello和Notion的内容索引
2. 定期整理：自动归档/归档过时的卡片和页面
3. 跨平台同步：重要信息同步到飞书统一查看
4. 待办聚合：聚合两个平台的待办事项统一管理

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| feishu-bitable | 存储整理后的索引和待办 |
| feishu-im-user-message | 发送整理提醒 |
| feishu-task | 创建待办任务 |

## 📝 Prompt 模板
```
你是我的效率工具整理助手。

帮我统一管理Trello和Notion：
1. 每月整理一次Trello和Notion：
   - 列出所有看板和页面，标注每个的用途
   - 归档超过30天未移动的卡片/过时页面
   - 清理空的列表/文件夹
2. 建立统一索引（飞书多维表格）：
   - 内容名称
   - 所在平台（Trello/Notion）
   - 链接
   - 重要程度
   - 最后更新时间
3. 每周一汇总两个平台的待办事项：
   - Trello：各个看板上标记为"待办"的卡片
   - Notion：带有TODO的任务页面
   - 统一展示给我，并可一键跳转到原位置
4. 提醒我在指定日期回顾/更新重要看板和页面

整理原则：保持简洁、去除杂物、确保重要内容可快速找到。
```

## ⏰ Cron 配置
```json
{ "schedule": "0 10 * * 1", "task": "trello_notion_organize" }
```
