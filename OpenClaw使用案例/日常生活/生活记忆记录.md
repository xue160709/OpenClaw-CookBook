# 生活记忆记录

**分类:** 日常生活  
**编号:** #60

## 🎯 痛点
- 很多美好的生活细节容易遗忘
- 想回顾某段经历但找不到记录
- 日记坚持不下来，太花时间

## ⚡ 核心功能
1. 语音记录：支持语音快速记录生活点滴
2. 照片关联：将照片与文字记录关联存储
3. 时间轴浏览：按时间线回顾生活经历
4. 标签检索：通过标签快速找到相关记录

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| feishu-bitable | 存储生活记录数据 |
| nodes | 调用手机摄像头拍照 |
| openai-whisper | 语音转文字记录 |
| feishu-doc-media | 关联照片到文档 |

## 📝 Prompt 模板
```
你是我的生活记忆记录助手。

帮我建立一个轻松无压力的生活记录系统：
1. 我可以随时通过语音或文字向你倾诉，你会帮我整理成简洁的生活记录
2. 每周五引导我做一次"一周回顾"：
   - 本周最开心的事是什么？
   - 本周最大的收获？
   - 本周有什么想记录的瞬间？
3. 记录自动按日期归档到飞书多维表格，包含：
   - 日期时间
   - 记录类型（感悟/事件/人物/地点）
   - 标签（可自定义）
   - 内容摘要
   - 相关照片（如有）
4. 支持按标签或日期检索回顾
5. 每月生成一篇"月度记忆卡片"，精选本月最值得纪念的5个瞬间

记录原则：轻松、不强制、尊重隐私。
```

## ⏰ Cron 配置
```json
{ "schedule": "0 18 * * 5", "task": "weekly_memory_review" }
```
