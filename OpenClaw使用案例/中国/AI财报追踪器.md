# AI 财报追踪器

**分类:** 中国特色用例  
**难度:** ⭐⭐

---

## 🎯 痛点

- A股财报信息分散，需要逐个查找
- 财报季关键时间点容易错过
- 手动整理数据耗时且容易出错
- 缺乏系统化的财报追踪和提醒机制

---

## ⚡ 核心功能

1. **财报获取** - 通过 AKShare 接口获取A股财报数据
2. **智能提醒** - 财报发布季自动提醒关注的公司
3. **数据存储** - 将财报关键指标存入多维表格
4. **异常预警** - 业绩大幅波动时自动预警
5. **定期报告** - 按周/月生成财报追踪报告

---

## 🛠️ 所需技能

| 技能 | 用途 |
|------|------|
| `exec` | 运行 AKShare Python 脚本获取数据 |
| `feishu_bitable` | 财报数据存储与管理 |
| `feishu_calendar` | 财报发布日程提醒 |
| `web_fetch` | 获取补充信息 |

---

## 📝 Prompt 模板

```
你是一个A股财报追踪助手，帮助我监控和分析上市公司财报。

核心能力：
1. 通过 AKShare 获取指定公司的财报数据
2. 追踪财报发布时间，准时提醒
3. 将关键指标存入多维表格
4. 分析业绩变化，识别异常波动

关注公司：[列出需要追踪的股票代码或名称]
```

## ⏰ Cron 配置示例

```json
{
  "schedule": "0 8 * * 1-5",
  "task": "earnings_calendar",
  "description": "每个交易日早上检查财报日历"
}
```

```json
{
  "schedule": "0 18 * * 4",
  "task": "earnings_summary",
  "description": "每周四生成周度财报汇总"
}
```
