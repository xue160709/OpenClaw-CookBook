# 飞书 AI 助手

**分类:** 中国特色用例  
**难度:** ⭐⭐

---

## 🎯 痛点

- 团队协作信息分散在多个平台
- 重复性的文档处理工作耗费时间
- 会议纪要需要手动整理
- 审批流程繁琐

---

## ⚡ 核心功能

1. **飞书机器人** - 部署为飞书机器人，响应消息
2. **文档自动化** - 自动生成、整理、格式化文档
3. **日程管理** - 创建会议、提醒、查询空闲时间
4. **任务管理** - 创建/更新飞书任务
5. **多维表格** - 数据录入、查询、统计

---

## 🛠️ 所需技能

| 技能 | 用途 |
|------|------|
| `feishu_im` | 飞书消息收发的 |
| `feishu_calendar` | 日程管理 |
| `feishu_task` | 任务管理 |
| `feishu_bitable` | 多维表格操作 |
| `feishu_doc` | 文档创建/更新 |

---

## ⚙️ 部署配置

```yaml
# openclaw.yaml
channels:
  - type: feishu
    bot_id: cli_xxxxx
    bot_secret: xxxxx
    verification_token: xxxxx
```

---

## 📝 Prompt 模板

```
你是一个飞书 AI 助手，负责协助团队高效工作。

核心能力：
1. 回答用户问题，提供信息查询
2. 帮助创建和更新飞书文档
3. 管理日程和会议安排
4. 创建和跟踪任务进度
5. 处理多维表格数据

沟通风格：专业、简洁、有帮助
```

## 💡 扩展用法

- **会议纪要自动化** - 飞书妙记 → 任务自动创建
- **周报汇总** - 从多维表格提取数据生成周报
- **审批提醒** - 自动提醒待审批事项
