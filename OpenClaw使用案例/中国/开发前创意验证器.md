# 开发前创意验证器

**分类:** 中国特色用例  
**难度:** ⭐⭐

---

## 🎯 痛点

- 开发前不了解市场需求和竞争情况
- 百度指数、微信指数等数据分散在不同平台
- 竞品分析耗时，缺乏系统性方法
- 难以快速验证创意可行性

---

## ⚡ 核心功能

1. **热度查询** - 百度指数、微信指数查询
2. **竞品扫描** - V2EX、知乎等平台竞品讨论分析
3. **需求验证** - 通过多维度数据验证创意可行性
4. **报告生成** - 一键生成创意验证报告
5. **持续追踪** - 持续监控关键词热度变化

---

## 🛠️ 所需技能

| 技能 | 用途 |
|------|------|
| `web_fetch` | 获取百度指数、微信指数页面 |
| `web_search` | 搜索V2EX、知乎等平台竞品信息 |
| `exec` | 运行数据分析脚本 |
| `feishu_bitable` | 验证数据存储 |

---

## 📝 Prompt 模板

```
你是一个创意验证助手，帮助我在开发前验证想法的可行性。

核心能力：
1. 查询百度指数、微信指数了解关键词热度
2. 在V2EX、知乎等平台搜索竞品讨论
3. 分析市场需求和竞争格局
4. 生成创意验证报告

请验证以下创意：[描述你的想法或产品]
重点关注：[指定需要分析的方向]
```

## ⏰ Cron 配置示例

```json
{
  "schedule": "0 9 * * 1",
  "task": "trend_check",
  "description": "每周一检查关注关键词的热度趋势"
}
```
