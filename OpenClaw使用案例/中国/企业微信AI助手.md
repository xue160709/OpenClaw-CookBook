# 企业微信 AI 助手

**分类:** 中国特色用例  
**难度:** ⭐⭐

---

## 🎯 痛点

- 企业微信与微信消息互通需要管理
- 客户资源难以系统化运营
- 内部审批流程繁琐
- 客服响应效率低

---

## ⚡ 核心功能

1. **企业微信机器人** - 消息收发、自动回复
2. **客户管理** - 客户标签、跟进记录
3. **群管理** - 自动拉人、踢人、欢迎语
4. **审批流** - 自动发起和处理审批
5. **消息存档** - 合规存档和检索

---

## 🛠️ 所需技能

| 技能 | 用途 |
|------|------|
| `wechat_work` | 企业微信消息 |
| `feishu_bitable` | 客户数据管理 |
| `memory` | 交互历史 |

---

## ⚙️ 部署配置

```yaml
# openclaw.yaml
channels:
  - type: wechat_work
    corp_id: your_corp_id
    agent_id: your_agent_id
    secret: your_secret
```

---

## 📝 Prompt 模板

```
你是一个企业微信 AI 助手，帮助企业进行客户运营和内部协作。

主要功能：
1. 客户咨询自动回复
2. 客户信息查询和更新
3. 群消息管理和通知
4. 内部流程咨询

注意：
- 遵守企业合规要求
- 保护客户隐私
- 复杂问题转人工
```

---

## 💡 扩展场景

- **sop 客群运营**：按标签自动发送欢迎语、跟进消息
- **裂变海报**：生成邀请海报、追踪邀请数据
- **客户画像**：汇总客户行为数据
