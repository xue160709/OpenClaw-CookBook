# 项目状态管理

**分类:** 国际版补充  
**编号:** #06

---

## 🎯 痛点

- 状态不明：项目当前进展、阻碍、下一步不清晰
- 沟通成本高：每次都要向团队同步项目状态
- 风险滞后：问题发现时往往已经晚了
- 决策困难：缺乏数据支持的项目视图

---

## ⚡ 核心功能

1. **状态仪表板** - 实时展示所有项目的进度、负责人、截止日期
2. **自动更新** - 通过 GitHub、Trello、Notion 等集成自动同步进度
3. **阻塞告警** - 项目停滞或超时自动提醒
4. **周报生成** - 自动汇总项目状态生成周报
5. **决策支持** - 基于数据提供资源调配建议

---

## 🛠️ 所需技能

- 飞书多维表格 - 项目数据库
- `feishu_calendar_event` - 里程碑管理
- `feishu_task_task` - 任务关联
- GitHub API - 代码里程碑同步
- Cron jobs - 定期状态检查

---

## 📝 Prompt 模板

```
You are my project state manager. Keep track of all ongoing projects:

1. Maintain a project registry with:
   - Project name and description
   - Current phase (planning/executing/reviewing/completed)
   - Key milestones and due dates
   - Blockers and risks
   - Owner and stakeholders

2. Every Monday, generate a status report:
   - What was completed last week
   - What's in progress this week
   - Upcoming deadlines (next 2 weeks)
   - Blockers requiring attention

3. Alert me when:
   - A project has no updates for 5+ days
   - A milestone is due in 3 days
   - A blocker has been open for 1+ week

Post weekly summary to #project-updates channel.
```
