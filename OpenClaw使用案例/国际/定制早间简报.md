# 定制早间简报

**分类:** 国际版补充  
**编号:** #03

---

## 🎯 痛点

- 信息过载：每天早上需要浏览多个新闻源、邮件、社交媒体，耗时很长
- 遗漏重点：手动筛选重要信息容易遗漏关键内容
- 时区混乱：跨时区追踪全球动态很困难
- 缺乏个性化：通用新闻简报往往不匹配个人关注领域

---

## ⚡ 核心功能

1. **多源聚合** - 整合新闻、社交媒体、邮件、天气预报等
2. **智能筛选** - 根据用户兴趣领域过滤噪音
3. **摘要生成** - 用 AI 生成简洁的要点摘要
4. **定时推送** - 每天早上自动生成并推送到指定渠道
5. **个性化学习** - 根据用户反馈调整内容偏好

---

## 🛠️ 所需技能

- `web_search` - 新闻和趋势搜索
- `web_fetch` - 网页内容抓取
- 飞书/Discord 集成 - 推送简报
- Cron jobs - 定时执行
- 飞书文档 - 存储简报历史

---

## 📝 Prompt 模板

```
You are my morning brief assistant. Every morning at 7 AM, generate a personalized brief:

1. Search for top stories in these areas: [用户关注的领域，如 AI、科技、创投]
2. Fetch key headlines from: [指定新闻源]
3. Check weather for [用户所在城市]
4. Review any urgent emails or calendar events

Format the brief as:
- 📰 Top 3 Stories (with 2-sentence summaries)
- 🔥 Trending Topics in [领域]
- 📅 Today's Priorities (calendar highlights)
- 🌤️ Weather: [城市] forecast

Post to #morning-brief channel. Keep it under 500 words.
Learn from my feedback to improve future briefs.
```

---

## ⏰ Cron 配置

```json
{
  "schedule": "0 7 * * *",
  "task": "morning_brief"
}
```
