# 个人 CLI 工具箱

**分类:** 工具开发  
**编号:** #06

---

## 🎯 痛点

作为独立开发者或设计师，你经常需要在终端执行各种重复性任务：项目初始化、代码格式化、文件处理、图片压缩等。你希望有一个统一的 CLI 工具箱，让 Agent 帮你执行这些任务，就像拥有一个「命令行助手」。

---

## ⚡ 核心功能

1. **自定义命令注册** - 将常用的命令行操作注册为简单指令
2. **一键执行** - 通过自然语言让 Agent 执行已注册的命令
3. **参数化运行** - 支持动态参数传递
4. **命令历史** - 记录和回溯执行历史

---

## 🛠️ 所需技能

| 技能 | 用途 |
|------|------|
| `exec` | 执行命令行操作 |
| `shell` 脚本 | 封装复杂操作 |
| 文件系统技能 | 读写配置文件 |

---

## 📝 Prompt 模板

```
你是我的个人 CLI 工具箱管理员。

功能：
1. 帮我注册和管理常用命令行工具
2. 执行我指定的任务

已注册的工具：
- `init-react` - 初始化 React 项目
- `compress-img` - 压缩图片文件
- `deploy-vps` - 部署到 VPS
- `git-clean` - 清理 Git 分支

当我需要执行任务时，调用对应的命令并返回结果。
```

---

## ⏰ Cron 配置

```
# 手动触发
Action: 用户请求 → 解析命令 → 执行 → 返回结果
```

---

## 💡 示例命令

```bash
# 初始化项目
/agent init-react my-app --template vite

# 压缩图片
/agent compress-img ./screenshots/* --quality 80

# 部署上线
/agent deploy-vps --env production
```
