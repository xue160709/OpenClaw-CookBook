# API 安全测试

**分类:** 安全监控  
**编号:** #32

## 🎯 痛点
API是系统对外的入口，可能存在注入攻击、越权访问、敏感信息泄露等安全漏洞。定期进行API安全测试是保障系统安全的重要手段。

## ⚡ 核心功能
1. **端点扫描** - 发现API端点
2. **认证测试** - 测试认证机制的强度
3. **注入检测** - 检测SQL注入、XSS等漏洞
4. **权限测试** - 测试越权访问
5. **敏感信息检测** - 检测敏感数据泄露
6. **报告生成** - 生成安全测试报告

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| Web安全 | API安全测试 |
| 网络诊断 | 端点发现 |
| 报告生成 | 生成安全报告 |
