# 网络延迟基准

**分类:** 安全监控  
**编号:** #26

## 🎯 痛点
在分布式系统或微服务架构中，网络延迟异常可能导致用户体验下降甚至业务中断。缺乏持续的网络性能监控手段，难以及时发现网络瓶颈或异常节点。

## ⚡ 核心功能
1. **多节点延迟检测** - 定期检测到多个目标节点的网络延迟
2. **阈值告警** - 当延迟超过预设阈值时触发告警
3. **历史趋势分析** - 记录延迟数据，分析趋势变化
4. **异常节点识别** - 自动识别延迟异常的节点
5. **报告生成** - 生成定期的网络性能报告

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| 网络诊断 | 检测节点延迟 |
| 消息通知 | 发送告警通知 |
| 日程/定时任务 | 定期执行检测任务 |

## 📝 Prompt 模板
```
你是一个网络性能监控助手。需要定期检测指定节点的网络延迟，并在延迟异常时告警。

检测目标：{target_nodes}
延迟阈值：{threshold}ms
检测间隔：{interval}

请执行网络延迟检测，当超过阈值时发送告警通知。
```
