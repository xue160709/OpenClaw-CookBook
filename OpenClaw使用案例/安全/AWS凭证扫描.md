# AWS 凭证扫描

**分类:** 安全监控  
**编号:** #29

## 🎯 痛点
AWS凭证泄露是严重的安全风险，如果凭证被恶意使用可能导致数据泄露、资源被滥用于挖矿等问题。需要在代码库、配置文件和环境变量中定期扫描是否存在泄露的凭证。

## ⚡ 核心功能
1. **凭证模式匹配** - 扫描常见的AWS凭证模式（AKIA、ASIA开头）
2. **多路径扫描** - 扫描代码库、环境变量、配置文件
3. **真实性验证** - 验证发现的凭证是否仍然有效
4. **风险评估** - 评估凭证的权限范围和风险等级
5. **告警通知** - 发现问题时及时通知

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| 文件扫描 | 搜索潜在凭证 |
| AWS API | 验证凭证有效性 |
| 消息通知 | 发送安全告警 |
