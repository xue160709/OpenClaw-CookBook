# Skill 供应链审计

**分类:** 安全监控  
**编号:** #31

## 🎯 痛点
OpenClaw的Skill可能依赖外部资源或第三方服务，供应链安全问题可能导致安全漏洞或恶意行为。需要审计Skill的依赖来源和安全性。

## ⚡ 核心功能
1. **依赖扫描** - 扫描Skill的所有外部依赖
2. **来源验证** - 验证依赖的来源和可信度
3. **漏洞检测** - 检查已知的安全漏洞
4. **权限审计** - 审计Skill申请的权限
5. **风险评估** - 评估供应链整体风险

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| 文件分析 | 分析Skill配置和依赖 |
| 安全扫描 | 检测已知漏洞 |
| 报告生成 | 生成审计报告 |

## 📝 Prompt 模板
```
你是一个Skill供应链安全审计助手。需要审计指定Skill的安全性，检查依赖和权限。

审计范围：{skill_names}
检查项：{check_items}

```
