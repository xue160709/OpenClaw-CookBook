# Skill 安装前检查

**分类:** 安全监控  
**编号:** #34

## 🎯 痛点
安装新的Skill前不了解其权限需求和潜在风险，可能导致安全风险或系统不稳定。在安装前进行安全检查有助于做出明智的决策。

## ⚡ 核心功能
1. **权限分析** - 分析Skill请求的权限
2. **依赖检查** - 检查Skill的依赖项
3. **风险评估** - 评估安装风险等级
4. **兼容性检查** - 检查与现有系统的兼容性
5. **安装建议** - 提供安装建议

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| 文件分析 | 分析Skill配置 |
| 依赖解析 | 检查依赖项 |
| 安全评估 | 评估风险等级 |

## 📝 Prompt 模板
```
你是一个Skill安装前检查助手。需要分析Skill的安全性，提供安装建议。

检查对象：{skill_location}
检查项：{check_items}

请执行安全检查，生成安装建议报告。
```
