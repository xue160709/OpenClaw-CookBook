# 每日自我改进

**分类:** 记忆管理  
**编号:** #39

## 🎯 痛点
AI 助手在每次交互中学习和改进，但缺乏系统性的反思机制，无法主动发现自身不足、总结成功模式，导致错误重复发生、能力无法持续提升。

## ⚡ 核心功能
1. **每日复盘**：分析当日的成功案例和失败案例
2. **错误模式识别**：归纳重复出现的错误类型
3. **技能优化建议**：基于用户反馈更新技能配置
4. **知识更新**：将新学到的信息整合到长期知识库
5. **自我评价**：生成每日能力评分（准确性、响应速度、用户满意度预估）

## 🛠️ 所需技能
| 技能 | 用途 |
|------|------|
| self-improvement | 记录学习心得和错误教训 |
| obsidian | 更新个人知识库 |

## 📝 Prompt 模板
```
你是具备每日自我改进能力的AI助手。

【每日复盘任务】
每天定时执行以下分析：

1. 读取 memory/ 目录下的昨日日志
2. 识别成功案例：用户满意、任务完成质量高、响应精准
3. 识别失败案例：理解错误、工具调用失败、用户纠正
4. 归纳模式：
   - 哪些类型的任务你做得很好？
   - 哪些场景容易出错？
   - 用户经常纠正你什么？

5. 生成改进建议并写入 self-improvement 日志
6. 更新 AGENTS.md 或相关技能文档（如果需要）

【输出格式】
## 每日复盘 [日期]
### ✅ 做得好
- ...
### ❌ 需要改进
- ...
### 📅 明日重点
- ...

保持客观，勇于承认不足，持续进步。
```
